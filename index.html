Yi Lu

I am a software engineer at Google.

I completed my PhD in computer science in 2020 at MIT, where I was advised by Prof. Sam Madden.

My thesis is on transaction processing in distributed OLTP databases.

Before my PhD, I received my M.Phil. degree from the Chinese University of Hong Kong in 2015 and my bachelor's degree from Harbin Institute of Technology, China in 2013.

Work Experience

  • F1 Query, Google, Sunnyvale, CA

    Software Engineer, 2020/07 - Present

    F1 Query is a distributed SQL query engine that powers more than 100 billion production SQL queries each day and supports critical users and applications at Google.

  • Realtime data, Facebook, Seattle, WA

    Software Engineer Intern, 2018/06 - 2018/08, Mentor: Wolfram Schulte

    Enriched the source language upm/stream (a DSL in Python) with new capabilities for authoring aggregators as monoids and composing them. Added translation of upm/stream to functional stylus (a CPP library). Extended functional stylus to make it failure tolerant (by introducing checkpointing and restartability).

  • Data Management Exploration and Mining Group, Microsoft Research, Redmond, WA

    Research Intern, 2016/05 - 2016/08, Mentor: Sudipto Das

    Worked on AutoIndexing project, which collects statistics from SQL Server and analyzes performance regression after implementing configurations recommended by database tuning advisor.
